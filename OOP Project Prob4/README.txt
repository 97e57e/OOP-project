계산기의 기능을 수행하는 프로그램입니다.

해야할일을 구분해보면 입력받은 문자열을 숫자와 부호로 나누기. 곱셈연산 먼저하기 입니다.
우선 문자열을 나누기 위해서 SplitInput이라는 함수를 만들었습니다. 이 함수에서는 입력받은 문자열을 받아옵니다. 입력받은 문자열을 앞에서부터 살펴봅니다. 띄어쓰기가 나왔으면 그 전까지 입력받은 문자들을 2차원 배열에 저장해줍니다. 모든 원소들이 나누어지면 몇개의 문자열들이 저장되었는지를 리턴 해 줍니다.(제대로된 리턴값이라면 항상 홀수 입니다.)

나누어진 문자열들을 숫자와 부호로 구분 해 줍니다. 짝수번째에 저장된 문자열들은 숫자, 홀수번째에 저장된 문자열들은 부호입니다. 따로 저장해 배열에 넣어주는데 숫자는 배열에 넣어줄때 아예 객체를 만들어 객체의 배열에 저장을 해줍니다.

그 다음엔 곱셉을 먼저 연산해주는 과정입니다. 부호의 배열을 살펴보면서 만약 '*'이 있다면 그 자리에 맞는 숫자와 그 다음 숫자를 연산해주고 한칸씩 앞으로 당깁니다.(예를들어 객체 배열엔 1, 2, 3, 4가 저장되어 있고 부호 배열엔 +, *, -가 저장되어있다면 1, 2*3, 4와 +, -가 저장됩니다.)이렇게 해서 모든 곱셈의 연산이 끝나면 +와 -는 우선순위가 같기 때문에 앞에서부터 차례차례연산 해주면 됩니다.
